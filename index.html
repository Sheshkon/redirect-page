AI algorithm

When all information about the board is known, AI determines all possible positions of the current tetromino on the board. Choose the best possible position via fitness function and transmits all the necessary moves by emulating keystrokes on the keyboard. Fitness function parameters:

Height
Maximum height of the grid.
Aggregate Height
The sum of the height of each column.
Cleared lines
Number of complete lines in the grid.
Number of holes
Empty space such that there is at least one tile in the same column above.
Bumpiness
Sum of the difference between heights of adjacent pairs of columns.
